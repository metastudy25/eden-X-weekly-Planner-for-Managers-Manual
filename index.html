📤 공유 코드로 원격 조회하기
공유 코드란? 학생이 「📤 공유 코드」 버튼으로 생성한 긴 텍스트 코드입니다. 이 코드 안에 학생의 플래너 전체 데이터가 담겨 있어, 어느 기기에서나 붙여넣기만 하면 학생 플래너를 조회할 수 있습니다.
1

학생에게 공유 코드 요청

학생에게 플래너 상단 「📤 공유 코드」 버튼을 눌러 코드를 생성하고, 「📋 복사」 후 카카오톡으로 전송해달라고 안내합니다.

코드가 매우 길게 보여도 그대로 전송하면 됩니다.
2

관리자 대시보드에서 붙여넣기

관리자 대시보드 하단 「공유 코드로 조회」 입력창에 받은 코드를 붙여넣고 「조회하기」를 클릭합니다.

🟠 화요일

공유 코드 또는 공용 PC 저장본으로 과목별 비율이 적절한지 확인. 수능 목표 대비 지나치게 치우친 과목 개별 피드백.

🔵 수요일

중간 점검

계획 대비 실천 여부 확인. 잔여 시간 현황 점검 후 간단 피드백 제공.

🟢 금요일

주간 마무리 상담

인쇄본 기반 1:1 면담. 이번 주 잘된 점·부족한 점 정리. 다음 주 방향 제시.

⚪ 수시

메모 내용 확인

"영어"가 아닌 "영어 — 단어암기"처럼 구체적으로 쓰고 있는지 점검. 막연한 계획은 구체화하도록 지도.

📌 월말

누적 데이터 분석

매주 인쇄본 모아 월간 공부 패턴 분석. 과목 비율이 성적 향상으로 연결되는지 점검 후 조정.

💬 상담 활용 팁
상담 상황활용 방법
계획표를 너무 빡빡하게 짠 학생 잔여 시간이 0분에 가까운지 확인. "여유 시간 15~20% 남기기" 지도.
특정 과목에만 시간이 몰린 학생 과목별 비율 표에서 특정 과목이 50% 이상인지 확인. 균형 조정 상담.
계획표와 실제 공부가 다른 학생 메모란을 구체적으로 쓰도록 지도. "무엇을, 얼마나" 명확히 기록하게 함.
저장을 안 하는 학생 학원 공용 PC 즐겨찾기 등록 + 등원 직후 저장 루틴 함께 만들어주기.
매주 같은 계획을 반복하는 학생 이전 계획 대비 달라진 점 질문. 메모란에 지난주 피드백 반영 여부 확인.

3

수정 방법

학생 추가: 마지막 항목 뒤에 쉼표 후 새 줄에 '이름':'비번' 추가
학생 삭제: 해당 줄 전체 삭제 (앞 줄 끝에 쉼표 확인)
비번 변경: '비번' 부분의 숫자만 수정

쉼표와 따옴표를 정확히 맞춰야 오류가 나지 않습니다.

⚠ 변경 후 반드시 새 비밀번호를 안전한 곳에 별도 보관하세요.

❓ 관리자 자주 묻는 질문
Q. 학생이 저장했는데 대시보드에 「저장됨」이 안 표시됩니다.▼
학생이 학원 공용 PC가 아닌 자신의 스마트폰이나 가정 PC에서 저장한 경우입니다. 저장 데이터는 해당 기기·브라우저에만 보관되므로 관리자 화면에는 나타나지 않습니다.

해결 방법: 학생에게 📤 공유 코드를 카카오톡으로 받아 「공유 코드로 조회」를 이용하세요.
Q. 관리자로 로그인이 안 됩니다.▼
로그인 화면 하단 「관리자로 로그인」 링크를 클릭하면 별도 비밀번호 입력창이 열립니다. 상단 이름·비밀번호 칸이 아닌, 열린 패널에 비밀번호를 입력해야 합니다.

비밀번호: eden2026
Q. 학생을 새로 추가하거나 비밀번호를 바꾸고 싶습니다.▼
「⚙ 학생·비번 관리」 탭을 참고해 HTML 파일을 수정한 후 GitHub에 재업로드하세요.
수정 내용은 1~2분 내에 자동 반영됩니다.
Q. 학생이 이름을 잘못 입력해서 로그인을 못 합니다.▼
이름은 HTML 파일에 등록된 그대로 입력해야 합니다. 박준혁(H)처럼 괄호 포함 여부 등을 학생에게 정확히 안내해주세요.

이름 형식을 단순화하고 싶으면 HTML 파일의 const STUDENTS 부분에서 이름 키를 수정할 수 있습니다.
Q. 플래너 링크(GitHub Pages)를 처음 만드는 방법이 궁금합니다.▼
① GitHub 계정 생성 (github.com 에서 무료 가입)
② 새 저장소(Repository) 생성 → 「Add file」 → Upload files로 HTML 파일 업로드
③ 저장소 Settings → Pages → Branch: main → Save
④ 몇 분 후 https://계정명.github.io/저장소명/파일명.html 형태로 접속 가능

이 링크를 학생들에게 카카오톡으로 공유하세요.
Q. 인쇄할 때 학생 플래너가 A4 한 장에 다 안 들어옵니다.▼
① 플래너 상단에서 「가로」 방향을 선택하세요.
② 인쇄 대화상자에서 가로 방향, 「페이지에 맞춤」을 선택하세요.
③ 크롬 브라우저에서 「추가 설정 → 배율 → 페이지에 맞춤」을 선택하면 가장 깔끔하게 출력됩니다.
